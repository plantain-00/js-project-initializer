# js-project-initializer
An tool to initialize a nodejs project or frontend project.

#### install

`npm i js-project-initializer -g`

#### usage

run `js-project-initialize`

#### features

- [x] UI component
- [x] Js Library
- [x] Nodejs Backend
- [x] Frontend
- [x] Nodejs Backend with Frontend
- [x] CLI
- [ ] electron project

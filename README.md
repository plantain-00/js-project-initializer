An tool to initialize a nodejs project or frontend project.

## install

`npm i js-project-initializer -g`

## usage

run `js-project-initialize`

## features

- [x] UI Component
- [x] Js Library
- [x] Nodejs Backend Program
- [x] Frontend Program
- [x] Nodejs Backend Program(with Frontend)
- [x] CLI Program
- [x] Electron Program

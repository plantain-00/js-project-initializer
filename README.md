An tool to initialize a nodejs project or frontend project.

## install

`yarn global add js-project-initializer`

## usage

run `js-project-initialize`

## options

key | description
--- | ---
-h,--help | Print this message.
-v,--version | Print the version

- [x] UI Component
- [x] Js Library
- [x] Nodejs Backend Program

- [x] Electron Program
